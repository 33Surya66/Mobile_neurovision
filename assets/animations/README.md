# Lottie Animations

This directory should contain the following Lottie animation files for the landing page:

1. `face_scan.json` - An animation of a face being scanned
2. `analytics.json` - An animation showing data visualization
3. `rocket.json` - A launch/rocket animation

You can find free Lottie animations at:
- [LottieFiles](https://lottiefiles.com/)
- [Lordicon](https://lordicon.com/)
- [Icons8](https://icons8.com/illustrations/illustration/lottielab-3d-astronaut)

Place the downloaded JSON files in this directory and update the filenames in `landing_page.dart` if necessary.

Example command to download a Lottie animation:
```bash
curl -o assets/animations/face_scan.json https://assets10.lottiefiles.com/packages/lf20_ab6y5ebd.json
```
